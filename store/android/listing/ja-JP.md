# Google Play ストア掲載情報 — 日本語

## アプリ名

```text
Spark ADHD習慣トラッカー
```

## 短い説明

```text
柔軟な習慣、小さな一歩、集中タイマー、見える達成をひとつのアプリで。
```

## 詳しい説明

```text
次の一歩を見える形に。

Sparkは、ADHDの特性と日によって変わるエネルギーを考えて作られた、柔軟な習慣・集中サポートアプリです。計画を小さな行動に変え、できたことを分かりやすく表示します。

今に合う行動を選ぶ
• エネルギー、使える時間、場所で候補を絞り込み
• 習慣ごとに小さめ・標準・チャレンジの行動を選択
• 選択肢を次の一歩ひとつだけに整理

前向きな継続を育てる
• 毎日または1日おきのペースを設定
• 達成の連続記録と、獲得した保護機能を利用
• 完了した行動とSparkポイントの内訳を確認

始める負担を減らす
• 2、5、10、25、50分の集中セッション
• 思いつきをすぐ記録し、整理はあとから
• ルーティンを一手順ずつ進め、外出準備の開始時刻を逆算
• 今日、集中、記録、進捗、ルーティンのウィジェット

初期設定でプライベート
最初のバージョンはアカウントやサーバーなしで動作します。習慣、メモ、ルーティン、履歴は端末内に残ります。広告や第三者の行動分析はありません。

Sparkは計画と自己管理のためのツールです。ADHDやその他の病気の診断、治療、治癒、予防を行うものではなく、専門家による支援の代わりにはなりません。
```

この翻訳は英語版と同じ無料・オフライン版を説明しています。
